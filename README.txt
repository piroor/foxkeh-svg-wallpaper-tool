SVG Wallpaper Tool について

概要

SVG Wallpaper Tool は、SVGのグラフィック要素をブラウザ上で自由にレイアウトするための JavaScript ライブラリです。
フォクすけの壁紙メーカーのような、利用者がインタラクティブにグラフィックを操作できるウェブサービスの制作をサポートします。

SVGって？

SVG は、イラストや図形の形と位置を記憶している画像形式のひとつで、次世代の Web 標準技術として注目されています。
形や位置が記憶されているので、画像の一部分だけを移動させたり、大きさを自由に変更することができ、拡大しても常になめらかな線で表示させることができます。
スクリプトを使って簡単に加工できるのも大きな特徴です。SVG Wallpaper Tool では、この技術を使って、素材を自分の好きなようにレイアウトできます。

機能

SVGの要素に対して下記の操作が可能です
▪配置
▪削除
▪移動
▪回転
▪拡大縮小
▪透明度変更
▪上下左右に反転
▪重なり順の変更

その他の機能
▪キャンパスサイズの変更
▪ドロップ＆ドラッグによる要素の追加
▪SVGファイルの書き出し

対応ブラウザ
下記ブラウザでの動作を確認しています。
▪Firefox 3.6+
▪Chrome 9+
▪Safari 5 (ドロップ＆ドラッグによる要素の追加には対応していません)

ライセンス

SVG Wallpaper Tool は、GPLv3とMITライセンスのデュアルライセンスで配布しています。ライセンスの範囲内で自由に利用・改変・再配布が可能です。
同梱されているSVGファイルのライセンスは「クリエイティブ・コモンズ 帰属-非営利 2.1 日本」です。
